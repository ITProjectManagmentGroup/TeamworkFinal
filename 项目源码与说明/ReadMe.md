README
======

#### 1 开发环境

IDEA 2020

Ubuntu 18.04

Java SDK 1.8

#### 2 技术栈

模板引擎 ：Thymeleaf

前端框架 ：Bootstrap

后端：Spring Boot+Gradle

数据库：Mysql

#### 2 实现功能

- 注册
- 登录：登陆后通过session的方式让后端识别操作者身份
- 学生管理台
	- 此页仅示意17个功能。
- 登录拦截器：以防用户即使不登录也能通过url的方式访问到管理台界面
- 解决了刷新时表单重复提交的问题
- 退出登录
